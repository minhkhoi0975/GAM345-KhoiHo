# Quick Sort vs Merge Sort in Worst Case of Quick Sort
reset
set terminal png size 1920,1080
set output 'worst_case_output_quicksort_vs_mergesort.png'
set title 'Average Completion Time of Quick Sort vs Merge Sort in Worst Case of Quick Sort'
